## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The website will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Image Sources

The website uses images from Unsplash. To ensure images feature black women, you can:

1. **Replace with specific Unsplash photos**: Visit [Unsplash](https://unsplash.com) and search for "black woman cleaning" or "professional cleaner"
2. **Use your own images**: Replace the image URLs in the components with your own hosted images
3. **Update image URLs**: Edit the image URLs in:
   - `src/components/Hero.jsx`
   - `src/components/Services.jsx`
   - `src/components/About.jsx`
   - `src/components/Testimonials.jsx`
